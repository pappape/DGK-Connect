# DGK-Connect
DGKサーバー専用接続ソフト
Releaseよりインストーラーをダウンロードしてインストールしてください。

# 使い方
1. インストールする
2. ソフトを開いて「接続」を押す。
3. Minecraft で「サーバーを追加」する
4. 下記の通りに設定する
   サーバー名: わかりやすいもの(自由)
   アドレス(サーバーごとに違います):
    DGK01: localhost:25565
    DGK02: localhost:25566
    DGK03: localhost:25567
    DGK04: localhost:25568
6. 遊ぶ
